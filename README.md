# Welcome to the Wellness app 👋

A mental wellness mobile app built with Expo React Native and Firebase.

## Users can
1. Create an account
2. Securely sign in
4. Log their mood
3. Track daily wellness activities
5. Enter journal entries


## Features
1. Firebase Authentication
   - Sign Up/In
   - Sign Out
2. Firestore Database
   - Add wellness activities
   - Delete activities
3. Mood Tracking


## Tech Stack 
- React Native
- Expo
- Expo Router
- Firebase Authentication
- Firebase Firestore
- TypeScript


## Get started

1. Clone the repository

   ```bash
   git clone <https://github.com/luthrasaara/evolvere-expo-app.git>
   ```

2. Run 

   ```bash
   npm install
   ```

5. Start the development server

   ```bash
   npx expo start
   ```

6. Run on: Andorid Simulator


## Firebase Setup
Create a Firebase project and add your Firebase configuration inside: firebase/config.ts

Enable:
1. Authentication: Email/Password 
Firestore Database
2. Authentication: Email/Password
Firestore Database
